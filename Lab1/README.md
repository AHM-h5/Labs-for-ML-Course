## What is Git?
Git is a version control system that helps developers track changes in their code over time. It allows users to save different versions of a project, collaborate with others, and revert to previous versions if needed.

## What is GitHub?
GitHub is an online platform that hosts Git repositories. It allows developers to store their projects in the cloud, collaborate with others, and manage their code using Git. GitHub also provides features such as repositories, commits, branches, and README files.
